# TypeScript

Ponder takes advantage of _ to create a great developer experience. 

- Ponder requires TypeScript >= v5.0.4 and viem >= v1.16.0.
- Types for indexing functions are powered by `ponder-env.d.ts`.
- Contract ABIs must be declared with `as const` for strict typing.

## Editor Support

Code editors also must follow the TypeScript version requirements. With VSCode, this can be changed with `TypeScript: Select TypeScript version..." from the command palette.
