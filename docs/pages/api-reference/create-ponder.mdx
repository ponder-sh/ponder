# `create-ponder`

This command line interface (CLI) bootstraps a TypeScript project directory with `package.json`, `ponder.config.ts`, `schema.graphql`, and other configuration files.

{/* prettier-ignore */}
<Tabs items={["pnpm", "yarn", "npm"]}>
  <Tab>
```bash
pnpm create ponder {...options}
```
</Tab>
  <Tab>
```bash
yarn create ponder {...options}
```
</Tab>
  <Tab>
```bash
npm create ponder@latest {...options}
```
</Tab>
</Tabs>

## Options

| option                      | description                                                              |
| :-------------------------- | :----------------------------------------------------------------------- |
| `--from-subgraph [path]`    | Bootstrap the project from a Graph Protocol subgraph repository          |
| `--from-etherscan [url]`    | Bootstrap the project from an Etherscan contract.                        |
| `--etherscan-api-key [key]` | An Etherscan API key to avoid rate limits with `--from-etherscan`.       |
| `--dir [path]`              | Create the Ponder project in the specified directory. Default: `ponder`. |
