# GraphQL query API

To learn more about GraphQL query language please refer to [this](https://graphql.org/learn/)

## Pagination

- Ponder server by default returns the first 100 rows.
- You can fetch at most 1000 rows in a single query.
- You can skip at most 5000 rows in a single query.
